# Data Analytics Internship – Task 1

## Objective
To perform data cleaning and exploratory data analysis on a real-world dataset and derive meaningful insights.

## Tools Used
- Microsoft Excel

## Data Cleaning
- Handled missing values using appropriate placeholders
- Standardized date format to YYYY-MM-DD
- Removed irrelevant unstructured columns
- Performed final data validation

## Analysis Performed
- Content type distribution (Movies vs TV Shows)
- Release year trend analysis
- Top contributing countries
- Ratings distribution

## Key Insights
- Movies dominate the content library
- Peak content release observed between 2017–2019
- United States is the leading content contributor
- Mature-rated content is most common

## Learning Outcome
This task helped me understand the importance of data cleaning and structured exploratory analysis before deriving insights.
