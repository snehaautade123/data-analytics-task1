# Customer Sales & Retention Insights Dashboard

## Dataset
This project uses the **Sales & Customer Insights dataset**, which contains transactional data including:
- Customer IDs
- Product IDs & Categories
- Purchase Frequency
- Average Order Value
- Region
- Churn Probability
- Lifetime Value
- Purchase behavior patterns
- Retention strategies

## Objective
The goal of this task is to clean, analyze, and visualize customer and sales data to generate actionable business insights. The focus is on:
- Understanding customer behavior
- Identifying high-value regions and categories
- Assessing churn probability and retention strategy effectiveness
- Highlighting peak purchase times

## Work Done
1. **Data Cleaning**
   - Handled missing values in key columns like Customer ID, Category, Lifetime Value, Churn Probability, and Preferred Purchase Times.
   - Standardized date formats to `YYYY-MM-DD` for compatibility and analysis.
   - Verified data types for all numeric, categorical, and date fields.
   - Removed duplicates to ensure accurate aggregation.

2. **Pivot Tables & Analysis**
   - Total Lifetime Value by Region
   - Average Churn Probability by Region
   - Customer Count by Most Frequent Category
   - Preferred Purchase Time Distribution
   - Retention Strategy Usage

3. **Dashboard Creation**
   - Built a professional Excel dashboard titled **“Customer Sales & Retention Insights Dashboard”**
   - Used **bar and column charts** for clear visual storytelling
   - Added **interactive slicers** for Region and Category
   - Summarized insights with concise, executive-ready headlines

## Key Insights
- North America & Europe generate the highest customer lifetime value.
- South America has the highest churn risk.
- Electronics and Clothing are the top revenue-driving categories.
- Loyalty Programs perform best for customer retention.
- Evening and Afternoon are peak purchase times.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers)
- Data cleaning & preprocessing in Excel

## Outcome
This dashboard provides **actionable insights** for business strategy, retention optimization, and marketing campaigns.  
It also serves as a foundation for **future predictive analytics and machine learning tasks** on the same dataset.
